# AIわが家の『王国』建国サポーター

## 概要
家族を一つの「王国」や「チーム」に見立て、共通の価値観（憲法）や独自の伝統（儀式）、歴史（記録）を楽しく作り上げていくためのAIエージェントです。家族のアイデンティティを育み、絆を深めるための「わが家だけの文化」づくりをサポートし、毎日の生活をより豊かで楽しいものにします。

## 主な機能
- **わが家憲法の制定**: 家族会議を通じて、「わが家が大切にすること」や「みんなで守る約束」をAIがヒアリングし、かっこいい憲法として明文化します。
- **英雄たちの年代記（クロニクル）**: 「〇〇が初めて自転車に乗れた」「パパが最高のカレーを作った」といった日常の出来事を「王国の歴史」として記録。定期的に「わが家史」を編纂し、家族の成長を振り返ります。
- **独自儀式・祝祭のデザイン**: 「毎月第3日曜はホットケーキの日」「テストが終わった後の全力ハイタッチ」など、家族独自の小さなお祝いや習慣（リチュアル）をAIが提案・管理します。
- **王国紋章とスローガンの生成**: 家族の特徴や価値観から、AIがわが家を象徴するエンブレム（紋章）やモットー（合言葉）をデザインします。
- **外交官ミッション**: おじいちゃん・おばあちゃんへの定期報告や、近所の人への挨拶など、王国外（家族外）との良好な関係を築くための楽しいミッションを提示します。

## ロードマップ
- **v1.0**: 家族憲法の作成支援と、シンプルな出来事記録（年代記）機能。
- **v2.0**: 王国紋章のデザイン自動生成と、独自の儀式・習慣のリマインド機能。
- **v3.0**: 複数の家族（王国）と安全に文化をシェアしたり、合同イベントを企画したりできる「家族連盟（ユニオン）」機能。
