# AIこども工作アドベンチャー (AI Kids' Craft Adventure)

## 概要
家にある空き箱、ペットボトル、トイレットペーパーの芯などの「廃材」が、AIの力で魔法のアイテムに生まれ変わる！
手元にある材料の写真を撮ると、AIがそれらを使って作れるクリエイティブな工作アイデアを提案し、完成までをガイドしてくれるアプリです。

## 主な機能
- **ざいりょうスキャン**: 家にある工作に使えそうな材料（段ボール、牛乳パック、ひも、ペットボトルキャップなど）をカメラで写すと、AIが種類と数を認識します。
- **ひらめきレシピ提案**: 認識した材料から、「宇宙船」「秘密基地」「動く動物」など、子どもの興味に合わせた工作の完成図をAIが複数提案します。
- **わくわく制作ガイド**: 選んだ工作の作り方を、子どもが理解しやすい言葉と図解（または動画）でステップバイステップで説明。難しい部分は「大人にお願いしてね」と安全への配慮も欠かしません。
- **バーチャル展示会**: 完成した作品の写真を撮ると、AIがかっこいい背景を合成。アプリ内の「デジタル美術館」に展示したり、家族に自慢の作品としてシェアできます。

## ロードマップ
- **v1.0**: 材料の画像認識と、シンプルな工作レシピ（テキストと静止画）の提案機能。
- **v2.0**: 音声ガイドによる制作サポートと、完成写真への背景合成エフェクト機能。
- **v3.0**: 他のユーザーの作品から刺激を受ける「こども発明家コミュニティ」と、必要なパーツ（モーターやLEDなど）を注文できる連携機能。
