# AIわが家のニュース放送局 (AI Family News Station)

## 概要
家族の何気ない日常や小さな出来事を、まるでプロのニュース番組やラジオ番組のように配信するAIエージェントです。スマートスピーカーやスマホを通じて、朝食時や夕食前の家族が集まる時間に「今日のわが家のニュース」を読み上げ、家族間のコミュニケーションを楽しく盛り上げます。

## 主な機能
- **デイリー・トピックス自動生成**: 家族の共有カレンダー、メッセージアプリでの会話、共有された写真から、AIがその日の「ニュース素材」を自動抽出して原稿を作成します。
- **ポジティブ・ニュース表彰**: 「〇〇ちゃんが逆上がりに成功！」「お父さんが夕飯を作ってくれた」など、家族のポジティブな行動を特派員レポート風に紹介し、お互いを褒め合うきっかけを作ります。
- **パーソナライズ・パーソナリティ**: 家族の好みに合わせて、落ち着いたニュースキャスター、元気なラジオDJ、可愛い動物キャラクターなど、AIの声や口調を自由に変更できます。
- **明日の「作戦会議」**: ニュースの最後に、明日の天気や予定を伝え、「明日の朝食は何にする？」といった家族で話し合うべきトピックを投げかけます。

## ロードマップ
- **v1.0**: テキストベースのニュース原稿生成と、音声合成による読み上げ機能。
- **v2.0**: 家族が投稿した写真を背景にした「ニュース映像（スライドショー形式）」の自動生成。
- **v3.0**: 家族の反応（笑い声や返答）を学習し、より家族のツボに刺さるネタ選びやジョークを交えるインタラクティブ放送機能。
