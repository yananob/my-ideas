# AIこども『まご』記者団

## 概要
AIが「編集長」となり、子どもが「記者」として祖父母にインタビューを行うことで、家族の歴史を紐解き、世代間の絆を深めるアプリです。子どもにとっては社会性や語彙力を養う「探究学習」になり、祖父母にとっては孫との深い交流を通じて自分の人生を肯定される「生きがい」に繋がります。

## 主な機能
- **インタビュー・ミッション生成**: AIが祖父母の出身地や生年代、家族の事前情報から、「子どもの頃の遊び」や「一番の冒険」など、興味深い「取材テーマ」を提案します。
- **リアルタイム取材サポート**: インタビュー中、AIが音声を聞きながら「それはどういうこと？」「もっと詳しく聞いてみて！」と、子どもが深掘りできるよう優しくアシストします。
- **家族の歴史新聞・自分史の自動生成**: 取材した音声や撮影した写真を元に、AIがプロのようなレイアウトの「家族新聞」や、祖父母の人生をまとめた「自分史」を自動的にデザイン・作成します。
- **タイムトラベル・ギャラリー**: インタビューで語られた昔のエピソードをAIが画像化。当時の風景や出来事を視覚的に再現し、家族で一緒に眺めることができます。
- **「ありがとう」の社説**: 取材の最後に、子どもが感じたことをAIが一緒に整理し、祖父母への感謝を込めた「社説（メッセージ）」として新聞に掲載します。

## ロードマップ
- **v1.0**: 音声録音とAIによる質問提案、テキストと写真によるシンプルな新聞生成機能。
- **v2.0**: AIによる画像生成（イメージビジュアル）を活用した、エピソード挿絵付きの豪華な自分史作成機能。
- **v3.0**: 離れた場所に住むいとこ同士で協力して一つの新聞を作る「共同通信社」機能と、完成した新聞を実際に印刷・製本して届けるサービスとの連携。
