# AIこども図書館・冒険ガイド

## 概要
地域の図書館を「探検のフィールド」に、本を「冒険の宝物」に変える、子どもの読書体験を最大化するためのAIエージェントです。子どもの移り変わる興味をキャッチし、最適な本との出会いを演出するだけでなく、読んだ後の対話を通じて思考力や想像力を育みます。

## 主な機能
- **わくわく読書カウンセリング**: AIが子どもと楽しく会話（または音声入力）し、「今、何が気になっているか」「どんなことにドキドキするか」を分析。今の気分や興味にぴったりの本を、図書館の蔵書から提案します。
- **図書館トレジャーマップ**: 提案された本が図書館のどの棚（分類番号）にあるかを、分かりやすいマップや「400番の棚へ行ってみよう！」といった冒険の指示として提示します。
- **読書パートナー対話**: 本を読んだ後、AIがその本の登場人物やテーマについて子どもに問いかけます。「もし君が主人公だったらどうした？」「一番驚いたのはどこ？」といった対話を通じて、深い理解とアウトプットを促します。
- **デジタル冒険通帳**: 読んだ本を記録すると、AIがその内容に基づいたオリジナルの「冒険スタンプ」やイラストを生成。読書量に応じてレベルが上がったり、新しい称号が手に入ったりするゲーミフィケーション要素を提供します。

## ロードマップ
- **v1.0**: 音声対話による本のレコメンド機能と、読書記録（スタンプ）機能。
- **v2.0**: 日本の図書館分類（NDC）に基づいた所在ガイド機能と、読後対話エンジンの搭載。
- **v3.0**: 地域の公共図書館の貸出状況とのリアルタイム連携、および読書感想文の作成サポート機能。
