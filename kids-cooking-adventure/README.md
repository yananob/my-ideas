# AIこども『お料理』大冒険 (AI Kids' Cooking Adventure)

## 概要
子どもが「自分でおいしいものが作れた！」という成功体験を積みながら、料理の楽しさと基本を学べるAI料理コーチアプリです。安全性を最優先に考慮し、年齢やスキルに合わせたステップバイステップのガイドを提供します。単なるレシピ集ではなく、食材の知識や調理の科学（なぜ？）、そして安全な道具の使い方を対話を通じて教える、子どもの自立心を育むためのパートナーです。

## 主な機能
- **キッズ・セーフ・レシピ**: 包丁を使わない「ちぎり・混ぜ料理」や、火を使わない「電子レンジ活用料理」など、子どもの成長段階に合わせた安全なレシピをAIが提案します。
- **AI対話型ハンズフリーガイド**: 調理中で手が離せなくても、AIが音声で次の手順を優しくガイド。「次はボウルに卵を割り入れよう」といった指示や、質問への回答をリアルタイムで行います。
- **「なぜなぜ」キッチン科学**: 「どうして玉ねぎを切ると涙が出るの？」「どうしてパンは膨らむの？」といった、料理中に生まれる疑問にAIがその場で科学的な根拠を交えて答えます。
- **AI盛り付けデザイナー**: 完成した料理をカメラで写すと、AIが「魔法の盛り付け」をアドバイス。「彩りに緑を足すとさらにおいしそうだよ！」といった提案で、表現力も育てます。
- **キッチン・セーフティ・チェック**: AIがカメラを通じて、包丁の持ち方や火の周りの安全を確認し、危険な兆候があれば優しくアドバイスします。

## ロードマップ
- **v1.0**: 年齢別セーフレシピの提供と、音声による基本的な調理ガイド機能。
- **v2.0**: 画像認識による「キッチン科学」解説と、盛り付けアドバイス機能の実装。
- **v3.0**: 家族の好みを学習した「わが家の味」カスタマイズ機能と、作った料理をデジタルアルバムにする「キッズ・シェフ・ジャーナル」機能。
