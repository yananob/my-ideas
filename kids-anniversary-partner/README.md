# AIこどもアニバーサリー・パートナー

## 概要
子どもの誕生から成人までの大切な節目（お宮参り、お食い初め、誕生日、七五三、入園・入学など）を、家族がゆとりを持って、かつ思い出深くお祝いできるようサポートするAIコンシェルジュです。準備の負担を軽減しながら、家族それぞれの価値観に合った「世界にひとつだけの記念日」をプロデュースします。

## 主な機能
- **パーソナライズ・アニバーサリー・カレンダー**: 子どもの生年月日に基づき、今後数年間にわたる行事スケジュールを自動生成。準備を始めるべきタイミングでリマインドします。
- **行事・風習ガイド**: 「お食い初めって何をするの？」「七五三の準備はいつから？」といった疑問にAIが回答。現代のライフスタイルに合わせたアレンジ案も提案します。
- **オーダーメイド・セレブレーション提案**: 子どもの今の興味（好きな動物、色、キャラクターなど）を取り入れたパーティーのテーマ、飾り付け、料理のレシピ、アクティビティをAIがプランニングします。
- **ギフト＆フォトプラン・コンシェルジュ**: 予算や好みに合わせて、最適なプレゼント候補や近隣のフォトスタジオ、お祝いができるレストランなどを提案します。
- **メモリー・メッセージ**: 準備中の親の気持ちや、当日のお祝いの様子を写真と共に記録。次の節目に「あの時はこうだったね」と振り返れるタイムトラベル機能を搭載。

## ロードマップ
- **v1.0**: 生年月日に基づく行事カレンダーと基本ガイド機能。
- **v2.0**: パーティーのテーマ提案とチェックリスト生成機能。
- **v3.0**: 地域情報（スタジオ、レストラン）の連携と、家族間での準備タスク共有機能。
