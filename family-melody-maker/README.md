# AIわが家のメロディ・メーカー

## 概要
日常の何気ない瞬間を家族だけの「歌」に変え、音楽を通じて家族の絆を深めるAIエージェントです。子どもの成長の記録や、家族の面白いエピソードをメロディに乗せて残すことで、思い出をより鮮やかに彩ります。

## 主な機能
- **エピソードから作詞・作曲**: 「今日、子どもが初めて一人で買い物に行けた」「パパが作ったカレーが激辛だった」などの出来事を話すと、AIがその時の感情にぴったりの歌詞とメロディを生成します。
- **家族の声で歌う**: 家族の声を短時間学習することで、AIが生成した歌を家族全員で合唱しているような音声で再生できます。遠くに住む祖父母へのボイスレターとしても活用できます。
- **シーン別BGM生成**: 「お片付けの歌」「歯磨きの歌」「おやすみ前の静かな曲」など、毎日のルーチンを楽しくするオリジナルBGMを即座に生成し、生活のリズム作りをサポートします。
- **音のタイムカプセル**: 作成した曲をカレンダー形式で保存。1年後や数年後に「あの時はこんなことがあったね」と、音楽と共に思い出を振り返ることができます。

## ロードマップ
- **v1.0**: テキスト入力からの作詞・作曲と、標準的なAI音声による歌唱機能。
- **v2.0**: 家族の音声学習（ボイスクローン）によるパーソナライズ歌唱機能の実装。
- **v3.0**: 楽器演奏アプリとの連携や、生成した曲をスマートスピーカーから自動再生する機能。
