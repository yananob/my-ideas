# AIエージェント・アプリアイデア集

個人でAIエージェントにつくってもらうアプリのアイデアを格納するリポジトリです。

## アイデア一覧

- [自動日報・感情分析トラッカー](./ai-mood-journal/) - デジタル活動ログから生産性とメンタルヘルスを可視化
- [AIパーソナルナレッジマネージャー (PKM)](./ai-pkm-manager/) - ローカルファイルやメモを統合管理する「第二の脳」
- [AIわが家のレシピ手帖](./ai-home-recipe-manager/) - 日々の記録から「わが家の味」を学習・提案するレシピ手帖
- [スマート学習カードジェネレーター](./ai-study-cards/) - ウェブ記事やドキュメントから学習カードを自動生成
- [AIファミリー・週末アドベンチャー](./family-weekend-planner/) - 子どもの興味や天気に合わせた週末プランを提案
- [AI子どもお手伝いクエスト](./kids-chore-quest/) - 日常のお手伝いをRPGのクエストに変えて楽しく習慣化
- [AI子ども美術館](./kids-art-museum/) - 子どもの作品をAIがキュレーションし、思い出を物語にする
- [AI魔法の読み聞かせ絵本](./ai-bedtime-storyteller/) - 子どもの日常を冒険に変える、世界に一つだけのパーソナライズ絵本
- [AIスクールプリント・オーガナイザー](./ai-school-print-organizer/) - 学校のプリントを撮影するだけで自動でカレンダー登録・管理
- [AIこども「なぜなぜ」研究室](./kids-curiosity-lab/) - 子どもの純粋な疑問を「研究クエスト」に変え、親子で学びを楽しむ
- [AI子どもの『夢』図鑑](./kids-dream-encyclopedia/) - 子どもの興味や将来の夢を記録し、探究心を育むパーソナル図鑑
- [AIこどもファッション・クローゼット](./kids-fashion-closet/) - 子どもの成長に合わせたサイズ管理と、毎日のコーディネートをAIがサポート
- [AIこども『ことば』の宝箱](./kids-word-treasure-box/) - 子どもの可愛い言い間違いや初めての言葉をAIが記録・分析するデジタル宝箱
- [AIこどもアニバーサリー・パートナー](./kids-anniversary-partner/) - 子どもの大切な節目をプロデュースし、家族の思い出作りをサポートするAIコンシェルジュ
- [AIこども習慣化サポーター](./kids-habit-formation-supporter/) - 子どもの日々のルーチンを楽しく習慣化し、達成感を育むサポートエージェント
- [AIこどもマネー・アドベンチャー](./kids-money-adventure/) - お金の使い方や貯め方を、AIとの対話を通じて楽しく学べるマネー教育アプリ
- [AI家族の防災クエスト](./family-disaster-quest/) - 家庭の防災対策をRPGのような冒険に変え、家族で楽しみながら備える
- [AIこども『きもち』探険隊](./kids-feelings-explorer/) - 子どもの感情理解と心のケアをサポートし、レジリエンスを育むAIエージェント

## ルール
- アイデアごとにディレクトリを作る
- 中身はマークダウンで、日本語で記述する
- アイデアの内容やユニークさ、ユーザーへの便利さにフォーカスして記述する。実装の詳細はいったん不要。
