# AIエージェント・アプリアイデア集

## ルール
- アイデアごとにディレクトリを作る
- 中身はマークダウンで、日本語で記述する
- アイデアの内容やユニークさ、ユーザーへの便利さにフォーカスして記述する。実装の詳細はいったん不要。
