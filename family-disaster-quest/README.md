# AI家族の防災クエスト

## 概要
「いつかやらなきゃ」と思いつつ後回しにしがちな家庭の防災対策を、家族みんなで楽しめる冒険（クエスト）に変えるアプリです。AIが家族構成や住んでいる地域、子どもの年齢に合わせたパーソナライズされた防災ミッションを提案し、楽しみながら「生き抜く力」を育みます。

## 主な機能
- **防災拠点（ホーム）カスタマイズ**: 家の図面や写真を読み込み、AIが「安全な場所」と「危険な場所」を判定。そこを「セーフゾーン」や「トラップエリア」としてゲームマップ化します。
- **サバイバル・パッキング**: 非常用持ち出し袋の準備を、RPGの装備を揃えるような体験にします。子どもの年齢に合わせて、お気に入りのおもちゃや好きなお菓子を「心の回復アイテム」として加える提案も行います。
- **おうち避難訓練ミッション**: AIが「地震発生！」などのナレーションを行い、家族で制限時間内に安全を確保するタイムアタック。完了後にAIが避難行動を評価し、メダルを授与します。
- **備蓄品ローリング・ストック・マスター**: 備蓄品の賞味期限を管理し、期限が近いものを「今日のスペシャル・メニュー」として活用するレシピをAIが提案。楽しみながら備蓄を更新できます。

## ロードマップ
- **v1.0**: 基本的な防災チェックリストのクエスト化と備蓄管理。
- **v2.0**: AR（拡張現実）を活用した、室内での危険箇所発見ゲーム。
- **v3.0**: 近隣の避難所までの道のりを歩いて「宝探し」のように確認するフィールドワーク機能。
