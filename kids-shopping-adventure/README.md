# AIこども『お買い物』アドベンチャー

## 概要
退屈になりがちなスーパーなどでの買い物を、子どもにとってわくわくする「冒険（クエスト）」に変えるアプリです。AIが買い物リストをミッション化し、旬の食材の見分け方やお金の仕組みを楽しく教えながら、親子の買い物を教育的で楽しい時間へとアップグレードします。

## 主な機能
- **お買い物クエスト生成**: 親が入力した「買うものリスト」を、AIが「魔法の薬の材料」や「秘密のパーティーの準備」といったストーリー仕立てのクエストに変換します。
- **AI食材スカウター**: スマートフォンのカメラをかざすと、AIが「どのトマトが一番甘そうか」「どの魚が新鮮か」をARや音声でガイドし、目利きの力を育てます。
- **ぴったし計算チャレンジ**: カゴに入れた商品の合計金額を予想したり、限られた予算内でどのお菓子を買うか選んだりする、AIによる算数・マネー教育機能。
- **食育ミニ知識**: 「この野菜はどうやって育つの？」「どこから来たの？」という子どもの疑問に、AIがその場でエージェントとして分かりやすく答えます。
- **冒険の証**: 買い物が終わると、その日の「お買い物レベル」や「見つけたお宝（買ったもの）」を記録したデジタルカードを発行し、達成感を演出します。

## ロードマップ
- **v1.0**: 買い物リストのミッション変換機能とシンプルな合計金額計算サポート。
- **v2.0**: 画像認識AIによる食材の鮮度・旬判別ガイド機能の追加。
- **v3.0**: 地域の店舗情報と連携した「ご当地食材発見クエスト」や、実際の家計簿アプリとの連携。
