# AIこども『発明家』チャレンジ (AI Kids' Invention Challenge)

## 概要
日常生活の中にある「ちょっとした不便」を「発明の種」に変え、AIと一緒に解決策を形にするアプリです。子どもの観察眼と創造力、そしてデザイン思考を遊びながら育むことを目的としています。

## 主な機能
- **不便見つけ隊（カメラ）**: 日常生活で「使いにくい」「困った」と思うものを写真に撮ると、AIがその原因を分析し、「発明のヒント」を提示します。
- **AI発明ブレストパートナー**: 提示されたヒントをもとに、AIとチャットしながら解決策（発明）のアイデアを膨らませます。「もし〜だったら？」「もっと〜にするには？」といった問いかけで思考を刺激します。
- **デジタル・プロトタイピング**: 考えた発明の仕組みを、AIが分かりやすい図解や3Dイメージとして生成。自分が考えたアイデアが形になるワクワク感を体験できます。
- **発明家ギルド（ギャラリー）**: 自分の発明をギャラリーに登録。他の小さな発明家たちのアイデアを見て、「すごい！」を送り合ったり、さらなる改良のヒントを得たりできます。

## ロードマップ
- **v1.0**: 不便の画像分析と、AIとの基本的なアイデア出し（チャット）機能。
- **v2.0**: 発明の図解・イメージ生成機能と、個人の発明コレクション機能。
- **v3.0**: ユーザー同士でアイデアを共有・改善し合えるギャラリー機能と、3Dプリンター等との連携ガイド。
