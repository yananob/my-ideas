# AIわが家の『哲学』カフェ

## 概要
家族で「正解のない問い」について楽しく深く対話することをサポートするAIエージェントです。子どもの思考力を育むとともに、親子の対話を通じてお互いの価値観を深く知り、家族の絆を深めるための「哲学対話（P4C）」の場を提供します。

## 主な機能
- **問いのインスピレーター**: 子どもの年齢や興味に合わせて、思わず考えたくなるような「大きな問い」（例：「しあわせって何？」「ルールはどうしてあるの？」）をAIが提案します。
- **対話のファシリテーター**: 家族全員が発言できるよう対話を進行し、「それはどういうこと？」「もし〜だったらどうなるかな？」といった問いかけで思考を深める手助けをします。
- **思考のビジュアルマップ**: 対話の内容をリアルタイムで整理し、家族それぞれの考え方の違いや共通点を可視化した「思考の地図」を生成します。
- **哲学の足跡（ジャーナル）**: 対話で生まれた素敵な言葉や新しい気づきを要約し、家族の成長記録として保存します。

## ロードマップ
- **v1.0**: 問いの提案と、基本的な対話ファシリテーション機能。
- **v2.0**: 思考のビジュアルマップ自動生成と、子どもの成長に合わせた問いの難易度調整機能。
- **v3.0**: 家族の日常の出来事から、その時にぴったりの「哲学的な問い」を抽出・提案するパーソナライズ機能。
