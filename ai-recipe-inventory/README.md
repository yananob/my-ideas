# AIレシピ＆在庫管理エージェント

## 概要
冷蔵庫の在庫をスマートに把握し、最適なレシピ提案から買い物リスト作成までを自動化する、キッチン専用のAIアシスタントです。日々の料理記録から「わが家の味」を学習し、使えば使うほど家庭に馴染む提案を行います。

## 主な機能
- **画像認識による食材登録**: 買ってきた食材や冷蔵庫の中身を写真に撮るだけで、AIが品目と数量を自動判別します。
- **賞味期限の管理と通知**: 登録された食材の賞味期限を管理し、期限が近いものを優先的に使った料理を提案します。
- **「わが家の味」学習機能**: 日々のレシピ入力から味の好みや定番食材をAIが分析。各家庭のスタイルに合わせた「ちょうどいい」レシピを提案します。
- **ご無沙汰レシピのリマインド**: 最近作っていないけれど以前は好評だったレシピを提案。献立のマンネリ化を防ぎ、家族の定番を大切にします。
- **パーソナライズされたレシピ提案**: 在庫食材、ユーザーの好み、摂取したい栄養素に基づいたレシピをAIが生成します。
- **スマート買い物リスト**: 在庫が切れたものや、レシピに必要な足りない食材を自動的に買い物リストに追加します。

## ロードマップ
- **v1.0**: 手動登録機能とAIによる基本レシピ提案。
- **v2.0**: カメラによる一括食材認識機能の実装。
- **v3.0**: 家族間での在庫共有機能と、ネットスーパーのカートへの自動追加連携。
