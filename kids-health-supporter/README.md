# AIこども健康サポーター

## 概要
子どもの日々の体調管理から、突然の発熱やケガなどの緊急時まで、AIが家族の健康をトータルにサポートする「デジタル家庭医アシスタント」です。症状の記録、ホームケアのアドバイス、そして病院での受診時に医師へ伝えるべき情報の整理を行い、子育て中の「どうしよう？」という不安を安心に変えます。

## 主な機能
- **症状タイムライン記録**: 発熱、咳、発疹、便の様子などを写真や音声で簡単に記録。AIが時系列に整理し、経過を一目で把握できるようにします。
- **受診用「サマリー」生成**: 病院に行く際、これまでの経過（いつから、どんな症状が、何回あったか）をAIが要約し、医師にそのまま見せられるメモを自動生成します。
- **安心ホームケア・ガイド**: 夜間の急な発熱や軽いケガに対し、信頼できる医療情報に基づいた「今すぐできる対処法」をAIがアドバイスします。
- **成長・予防接種スケジューラー**: 身長・体重の記録と成長曲線の自動描画、忘れがちな予防接種のスケジュール管理とリマインドを行います。
- **お薬手帳＆服用リマインダー**: 処方された薬を写真で取り込み、飲み忘れ防止のアラートや、副作用の有無などの経過記録をサポートします。

## ロードマップ
- **v1.0**: 症状のタイムライン記録と写真保存、簡単な受診用サマリー生成機能。
- **v2.0**: 成長記録のグラフ化と、信頼できるガイドラインに基づくホームケアアドバイス機能。
- **v3.0**: 予防接種スケジュール管理と、地域の小児科・夜間救急情報の連携。
