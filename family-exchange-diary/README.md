# AIわが家の「きずな」交換日記 (AI Family Connection Exchange Diary)

## 概要
家族の間でやり取りする交換日記にAIが「ファシリテーター」として参加し、普段なかなか言えない感謝の気持ちや、何気ない日常の発見を引き出し、家族の絆を深めるアプリです。忙しい毎日の中で途切れがちな家族のコミュニケーションを、AIが優しくサポートします。

## 主な機能
- **AIファシリテーション**: 「今日、お母さんが作ってくれた料理で一番好きだったのは何？」「お父さんの自慢できるところを1つ教えて」など、AIが家族一人ひとりに合わせた「お題」を提案し、書き込みを促します。
- **きずな要約・フィードバック**: 1週間分の書き込みをAIが分析。「今週、家族の間で『ありがとう』が15回ありました」「〇〇ちゃんは、最近算数を頑張っているようです」といったポジティブなフィードバックを届けます。
- **AI「家族」の参加**: ペットや、亡くなった祖父母、あるいは空想のキャラクターとしてAIが日記に参加。家族の会話にユーモアや癒しを加えます。
- **思い出アーカイブ**: 書き込まれた日記をAIが編集し、月ごとに「わが家の思い出マガジン」を自動生成。写真と言葉を組み合わせたデジタル冊子として保存できます。

## ロードマップ
- **v1.0**: テキストベースの交換日記機能と、AIによる日替わりの質問（お題）提案機能。
- **v2.0**: 写真や音声メッセージの投稿対応と、週ごとの感情・トピック要約レポート機能。
- **v3.0**: 1年間の記録を1冊の本にする「AI編集・フォトブック注文」サービスとの連携。
