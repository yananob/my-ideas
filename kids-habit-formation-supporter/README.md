# AIこども習慣化サポーター

## 概要
子どもの日々のルーチンや目標を楽しく習慣化するためのアプリです。AIが達成を褒めてくれたり、やるべきことを優しくリマインドしたりすることで、子どもの自律性を育み、家族のコミュニケーションを活性化します。

## 主な機能
- **AI達成お祝い**: タスクをチェックすると、AIがその頑張りを具体的に褒めてくれたり、画面上で楽しいエフェクトが表示されたりして、高い達成感を提供します。
- **インテリジェント・リマインド**: 予定の時間を過ぎてもチェックがない場合、AIが「まだかな？」「一緒に頑張ろう」と優しくリマインド通知を送ります。
- **週次振り返りレポート**: 1週間の達成状況をAIがまとめ、頑張ったポイントや成長した点を親子で振り返ることができるレポートを生成します。
- **ペアレンタル管理**: 保護者がタスク（宿題、歯磨き、片付けなど）を自由に設定・調整でき、子どもの成長に合わせた目標管理が可能です。

## ロードマップ
- **v1.0**: 基本的なタスク管理、チェック機能、AIによる簡単なコメント。
- **v2.0**: リマインド通知、週次レポートの生成、カスタムスタンプ/エフェクト機能。
- **v3.0**: 継続日数に応じたキャラクターの成長や、家族間での励ましメッセージ機能。
