# AIこども『世界旅行』冒険記 (AI Kids World Traveler)

## 概要
家にいながら世界中の国々をバーチャルに冒険し、現地の文化、言葉、食べ物、歴史をAIガイドと一緒に楽しく学べるアプリです。Google Earthやストリートビューと連携し、AIがその土地の歴史上の人物や精霊になりきってガイドすることで、子どもの国際感覚と探究心を養います。

## 主な機能
- **AI現地ガイドとの対話**: 選んだ国のキャラクター（例：フランスの「エッフェルくん」、エジプトの「スフィンクスさん」）が、その土地の挨拶や文化、面白い歴史をクイズ形式で教えてくれます。
- **バーチャル・食べ歩きミッション**: 「パリのパン屋さんでクロワッサンを探そう」「イタリアの市場で一番大きなチーズを見つけよう」といったミッションを、ストリートビューと連動してクリアしていきます。
- **AI翻訳魔法の杖**: スマホをかざすと、画面上の外国語が翻訳されるだけでなく、AIが現地の発音で読み上げてくれ、簡単な会話の練習ができます。
- **世界に一つの『冒険パスポート』**: 訪れた国で学んだことや撮影したスクリーンショットが、デジタルのスタンプ帳や日記として記録され、自分だけの世界旅行記が完成します。

## ロードマップ
- **v1.0**: 5カ国のバーチャルツアー機能と、AIキャラクターによる基本ガイド・クイズ機能。
- **v2.0**: ストリートビューを活用した探索ミッションと、音声認識による簡単な多言語会話練習機能。
- **v3.0**: 家族や友達とオンラインで一緒に冒険できるマルチプレイ機能と、現地の特産品が実際に届く「お取り寄せ連携」の実装。
