# 自動日報・感情分析トラッカー

## 概要
日々のデジタル活動ログをAIが解析し、あなたの生産性とメンタルヘルスを可視化するセルフケア・ツールです。

## 主な機能
- **自動活動ログ収集**: Google Calendar、Slack、GitHubなどのAPIと連携し、一日の活動を自動的にまとめます。
- **AI感情分析**: 日記やチャットの投稿内容から、その日の気分の変化をポジティブ/ネガティブの両面で分析します。
- **自動日報生成**: 「今日は何をしたか」をAIが1分で読める形式にまとめ、振り返りを容易にします。
- **ウェルビーイングのアドバイス**: 「最近深夜の作業が多いようです。明日は午前中を休みにしませんか？」といったパーソナライズされた提案を行います。

## ロードマップ
- **v1.0**: カレンダー連携とAIによる感情分析・日記機能。
- **v2.0**: 主要なビジネスツール（Slack, GitHub等）との連携拡大。
- **v3.0**: ウェアラブルデバイス（Apple Watch等）のデータを取り込んだ総合的な体調管理。
