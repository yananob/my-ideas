# AIパーソナルナレッジマネージャー (PKM)

## 概要
個人のローカルファイル、メモ、ブックマークを統合し、AIが文脈を理解して整理・検索を支援するツールです。大量の情報を抱える現代人の「第二の脳」として機能します。

## 主な機能
- **ローカルファイルの自動インデックス**: 指定したフォルダ内のテキスト、PDF、Markdownを自動でスキャンし、ベクトル化します。
- **自然言語による横断検索**: 「先月のプロジェクトの予算案はどこ？」といった曖昧な質問でファイルを見つけ出します。
- **関連ドキュメントの自動リンク提案**: 現在書いているメモに関連する過去の記録をAIが自動で提示します。
- **内容の要約とQA**: 長いドキュメントの内容を要約したり、その内容についてチャット形式で質問したりできます。

## ロードマップ
- **v1.0**: 基本的なテキストファイルのインデックスとセマンティック検索。
- **v2.0**: OCRを搭載し、画像やPDFスキャンの解析に対応。
- **v3.0**: ブラウザ拡張機能と連携し、閲覧履歴やブックマークも統合。
