# AI中高生『推し活』・趣味探究バディ (AI Teen Hobby & Passion Connect)

## 概要
中高生が熱中している趣味や「推し活」を、単なる娯楽としてだけでなく、自己探究やスキル向上、さらには家族との良好なコミュニケーションのきっかけに変えるためのAIエージェントです。多感な時期にある中高生の情熱を肯定的にサポートし、親が子どもの興味を理解し、尊重するための架け橋となります。

## 主な機能
- **「推し」プレゼン・シミュレーター**: 自分の好きなこと（推し）について、AIに対してプレゼンする練習ができます。AIは「聞き上手なファン」や「少し知識のある友人」になりきり、対話を通じて本人の言語化能力や論理的思考力を高めます。
- **趣味から広がる学びナビ**: アニメ、ゲーム、スポーツなど、本人の興味関心から関連する歴史、科学、技術、言語などの知識をさりげなく提案します（例：ゲームの背景にある神話の解説、アニメ制作の裏側にあるデジタル技術など）。
- **「推し活」家計簿・スケジュール**: 限られたお小遣いや時間をどう使うか、AIと相談しながら計画を立てることで、自律的な管理能力を養います。
- **ペアレンツ・インサイト（ポジティブ解説）**: 親に対し、子どもが何に熱中しているのかを「なぜそれが素晴らしいのか」「そこからどんな能力が育っているのか」というポジティブな視点で解説・報告します。共通の話題作りのためのヒントも提供します。

## ロードマップ
- **v1.0**: 趣味・推し活の記録機能と、AIによる対話型プレゼン練習機能。
- **v2.0**: 趣味に関連した学習コンテンツの自動レコメンド機能。
- **v3.0**: 予算・時間管理サポート機能と、親子間の「興味の共有」を支援するダッシュボード。
