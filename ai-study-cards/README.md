# スマート学習カードジェネレーター

## 概要
ウェブ記事、PDF、技術ドキュメントから、記憶すべき重要なポイントをAIが自動抽出し、効率的な学習（分散学習）をサポートするカードを作成します。

## 主な機能
- **ワンクリック抽出**: ブラウザ拡張機能を使用して、読んでいるページの重要箇所を即座にAIへ送ります。
- **Q&A自動生成**: 選択したテキストに基づき、記憶に定着しやすい形式の問いと答えをAIが作成します。
- **Anki連携**: 生成されたカードを .apkg 形式でエクスポートし、既存の学習アプリ（Anki等）に取り込めます。
- **難易度調整**: 初心者向け、専門家向けなど、ユーザーのレベルに合わせて抽出する情報の粒度を調整できます。

## ロードマップ
- **v1.0**: ブラウザでのテキスト選択からカード生成・エクスポートまで。
- **v2.0**: YouTube動画の文字起こしやPDFファイルからの自動生成に対応。
- **v3.0**: 独自の分散学習エンジンを搭載し、アプリ内で完結する学習体験の提供。
