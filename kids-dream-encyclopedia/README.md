# AI子どもの『夢』図鑑 (AI Kids' Dream Encyclopedia)

## 概要
子どもがふとした瞬間に口にする「将来の夢」や、今ハマっている「大好きなこと」を記録し、AIがそれを深めるための「探究クエスト」や関連情報を提案するアプリです。成長とともに変化する興味の軌跡を可視化し、親子の対話を豊かにします。

## 主な機能
- **夢・興味のクイック記録**: 「宇宙飛行士になりたい」「ダンゴムシに興味がある」など、音声や写真で手軽に記録。その時の子どもの熱量をそのまま残せます。
- **AI探究ガイド**: 登録された興味に基づき、近所の博物館の展示、おすすめの本、自宅でできる簡単な実験などをAIがパーソナライズして提案します。
- **成長の軌跡（ドリームマップ）**: 興味がどう移り変わってきたかを年表やマインドマップ形式で可視化。数年後に振り返ることで、子どもの個性の根源を発見できます。
- **応援メッセージ生成**: 子どもの興味に関連する分野の専門家（AIキャラクター）から、その子の活動を褒めたり、さらに好奇心を刺激したりするメッセージが届きます。

## ロードマップ
- **v1.0**: 興味・夢のテキスト記録機能、AIによる関連情報（本・場所）の基本的な提案、シンプルなタイムライン表示。
- **v2.0**: 音声入力・画像解析による記録の高度化、AIによる具体的な「探究クエスト」の生成、家族間での共有・コメント機能。
- **v3.0**: 地域の教育イベントとの自動連携、過去の興味データに基づいた進路や習い事の適性アドバイス、大人になった時のための「タイムカプセル」エクスポート機能。
