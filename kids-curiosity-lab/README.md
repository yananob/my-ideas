# AIこども「なぜなぜ」研究室

## 概要
子どもの日常的な「なぜ？」「どうして？」という疑問を、学びと発見の冒険に変えるアプリです。親が答えに窮するような質問に対しても、AIが子どもの年齢に合わせた分かりやすい解説と、親子で一緒に取り組める「小さな実験・観察クエスト」を提案します。

## 主な機能
- **なぜなぜボイスログ**: 子どもが発した疑問をその場で録音・記録。AIがその瞬間の好奇心を逃さずキャッチします。
- **年齢別ステップ解説**: 3歳児には擬人化したお話で、小学生には科学的な根拠を交えて、AIが理解度に合わせて解説を生成します。
- **「やってみよう！」クエスト**: 解説だけで終わらず、「公園でアリの巣を探してみよう」「コップに水を入れて光を当ててみよう」といった、家にあるものでできる簡単な検証活動をAIが提案します。
- **わくわく研究レポート**: 実験の結果や気づいたことを写真や音声で記録すると、AIが世界に一つだけの「研究レポート」としてまとめ、図鑑のような形式でコレクションできます。

## ロードマップ
- **v1.0**: 疑問の記録とAIによる解説・クエスト提案機能。
- **v2.0**: 写真付き研究レポートの生成とPDF保存機能。
- **v3.0**: 同じ疑問を持った他の子どもたちの発見を共有できる「こども学会」機能（プライバシー配慮型）。
