# AI子ども美術館 (AI Kids' Art Museum)

## 概要
日々増えていく子どもの絵や工作を、AIがプロのキュレーターのように整理・解説し、家族の宝物として永久保存するアプリです。ただの画像保存ではなく、作品に込められた子どもの意図を汲み取り、物語として再構成します。

## 主な機能
- **AI作品キュレーション**: スマホで写真を撮るだけで、AIが作品のテーマ、使用された色、描かれている対象を分析し、自動でタグ付けと分類を行います。
- **おしゃべり音声ガイド**: AIが作品の解説文を生成し、美術館の音声ガイドのように再生します。「この力強い筆致は、当時の作者の溢れんばかりのエネルギーを表現しています」といった本格的な解説を楽しめます。
- **物語への自動変換**: 複数の作品を選んで「物語にして」と頼むと、AIがそれらの絵を挿絵としたオリジナル絵本を作成します。
- **メタバース・ファミリーギャラリー**: 保存した作品が飾られた仮想美術館を生成。遠くに住む祖父母を招待して、一緒に作品を鑑賞しながらボイスチャットで思い出を語り合えます。

## ロードマップ
- **v1.0**: 作品のアップロード、AIによる自動タグ付け、簡易的な解説生成機能。
- **v2.0**: 音声ガイド機能と、作品をベースにした絵本（PDF/動画）生成機能。
- **v3.0**: VR/メタバース空間での展示機能と、家族間での共同キュレーション機能。
