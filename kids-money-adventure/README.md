# AIこどもマネー・アドベンチャー

## 概要
お金の役割や価値、管理方法を、AIとの対話やシミュレーションを通じて楽しく学べる教育アプリです。おこづかいの管理から、疑似的な投資体験、社会貢献まで、子どもの発達段階に合わせた「マネー・リテラシー」を育みます。

## 主な機能
- **AIおこづかい帳**: 使ったお金や貯めたお金をAIに報告すると、AIが「賢い買い物ができたね！」や「あと少しで目標のゲームが買えるよ」と励ましてくれます。
- **おしごと市場**: 家の中での特別なお手伝い（例：洗車、窓拭き）を「おしごと」として登録。達成すると、AIが発行するデジタル通貨（または実際の日本円）を受け取れます。
- **夢の貯金箱**: 欲しいものを登録すると、AIがそれに向けての貯金プランを提案。節約のヒントや、誘惑に負けそうな時のアドバイスをくれます。
- **キッズ・マーケット・シミュレーター**: 社会の仕組みを学ぶために、AIが店主や客となって、疑似的な売買体験や投資体験（例：おもちゃの会社の株を買う）をガイドします。

## ロードマップ
- **v1.0**: AIおこづかい帳と夢の貯金箱機能の実装。
- **v2.0**: おしごと市場と、完了報告のAI画像認識機能。
- **v3.0**: キッズ・マーケット・シミュレーターと、複数ユーザー間での物ブツ交換・寄付体験機能。
