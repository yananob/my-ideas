# AIこども『いきもの』観察バディ (AI Kids Nature Buddy)

## 概要
公園や道端で見つけた虫、花、鳥などにスマホをかざすと、AIがその生き物になりきって話しかけてくれる、対話型の自然観察アプリです。ただの名前調べ（同定）にとどまらず、生き物との「対話」を通じて、子どもの好奇心と生命への共感を育みます。

## 主な機能
- **いきもの対話・AR同定**: カメラを向けるとAIが生き物を特定。その生き物のキャラクター（例：元気なセミの「セミ之助」、おっとりした「たんぽぽお姉さん」）が登場し、ARで画面上に現れて自分の生態や秘密を教えてくれます。
- **観察ミッション**: 「このチョウがどこに行くか、3メートル離れて追いかけてみよう」「この石をそっとひっくり返して、誰が住んでいるか探してみよう」など、発見を深める具体的なアクションをAIが提案します。
- **世界に一つの『いきもの記録』**: 撮影した写真と、AIとの会話内容を元に、子どもが主人公の「観察物語」をAIが自動生成。オリジナルのデジタル図鑑として保存されます。
- **わが家の『いきもの地図』**: 家族で見つけた生き物を、地図上に記録。季節ごとにどんな生き物が近所に現れるかを家族で共有し、お散歩をより楽しくします。

## ロードマップ
- **v1.0**: 生き物の自動同定機能と、テキストベースの「いきもの対話」機能の実装。
- **v2.0**: ARキャラクターの表示、音声読み上げ機能、観察ミッションの自動生成。
- **v3.0**: 家族や友達との図鑑共有機能、季節や場所に基づいた「珍しい生き物出現予想」アラートの実装。
