# AI中高生『みらい』ナビゲーター (AI Teen Future Navigator)

## 概要
思春期を迎え、自立心が芽生える中高生とその保護者のためのAIエージェントです。勉強、部活、趣味に忙しい中高生の自己管理をサポートし、将来のキャリアや興味を一緒に探究する「伴走者」となります。同時に、親が過干渉にならずに子どもの状況を把握し、必要な時に適切なサポートができるよう、親子間の程よい距離感を保つ「通訳・ブリッジ」としての役割も果たします。

## 主な機能
- **セルフマネジメント・バディ**: 試験日程や提出物、部活動の予定を管理。AIが「今はこれをやる時間だよ」と優しくリマインドし、本人の自立的な行動を促します。
- **みらい探究ダイアログ**: 「将来何をしたいか」という漠然とした問いに対し、日々の興味関心から関連する職業や学問分野を提案し、AIと対話しながら自分のキャリアイメージを膨らませます。
- **ペアレンツ・ブリーフィング**: 親には細かな行動ではなく、「今は試験前で集中しているようです」「最近〇〇に興味を持っているようです」といった、子どもの「状態」や「関心の方向性」を要約して伝えます。
- **リスペクト・メッセージ**: 親から子へ、直接言うと煙たがられそうな労いや応援の言葉を、AIが最適なタイミングで（例えば夜遅くまで勉強している時などに）自然な形で届けます。

## ロードマップ
- **v1.0**: スケジュール管理と学習リマインド機能、キャリア興味のタグ付け。
- **v2.0**: 興味のある分野に関する最新ニュースや大学・職業情報のレコメンド機能。
- **v3.0**: 親子間のコミュニケーションを円滑にする「伝え方アドバイス」機能や、プライバシーに配慮した進捗共有システム。
