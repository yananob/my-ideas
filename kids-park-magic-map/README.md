# AIこども『公園』魔法の地図 (AI Kids' Park Magic Map)

## 概要
公園の遊具や自然をファンタジー世界の舞台に見立て、いつもの公園を壮大な冒険のフィールドに変えるアプリです。AIが身近な環境をクリエイティブに再解釈することで、子どもの想像力を刺激し、屋外での運動をより楽しく、主体的なものにします。

## 主な機能
- **公園スキャン & マップ生成**: 公園にある遊具（すべり台、ブランコ、砂場など）をスマホで撮影・登録すると、AIが「勇者の滝」「天空のブランコ」「未知の砂漠」といったファンタジーな名称と設定を付与した冒険マップを自動生成します。
- **冒険クエスト発行**: 「勇者の滝を3回滑り降りて、水の精霊の加護を受けよ」「未知の砂漠に隠された宝（きれいな石や葉っぱ）を3つ見つけよ」といった、遊具や自然環境を活用したクエストをAIがその場で提案します。
- **AR冒険ログ**: 特定の場所で写真を撮ると、AIがファンタジー風のエフェクト（伝説の剣、魔法のオーラ、仲間のモンスターなど）をリアルタイムで合成。冒険の思い出を「冒険日誌」としてデジタル保存できます。
- **安全見守り & 休憩ガイド**: 子どもの活動量や周囲の状況を見守り、「そろそろHP（体力）が減ってきたよ。お水（ポーション）を飲んで休憩しよう」と、遊びに夢中な子どもを適切なタイミングで休息へ誘導します。

## ロードマップ
- **v1.0**: 公園の遊具登録機能と、テキストベースのクエスト生成機能のリリース。
- **v2.0**: カメラを通じたARエフェクト合成機能と、写真付き冒険日誌の自動作成機能の実装。
- **v3.0**: 複数の公園を繋いだ「ワールドマップ」の構築や、近隣のユーザー（冒険者）と協力して強大なボス（架空の存在）に挑むレイドクエスト機能の追加。
