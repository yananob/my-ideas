# AIファミリー・週末アドベンチャー

## 概要
子どもの年齢、興味、現在の天気、そして親の疲れ具合（！）までを考慮して、家族全員が楽しめる最高の週末プランを提案するAIコンシェルジュです。

## 主な機能
- **パーソナライズされたプラン提案**: 「恐竜が好き」「水遊びがしたい」「最近は工作にハマっている」といった子どもの興味関心に合わせて、近場のイベントや公園、博物館などを提案します。
- **おまかせ準備リスト**: 行き先が決まると、その日の気温やアクティビティに必要な持ち物リスト（着替え、日焼け止め、おやつ、レジャーシート等）を自動生成します。
- **雨の日バックアップ**: 予定していた屋外イベントが雨で中止になりそうな場合、AIが即座に近隣の屋内施設や、自宅で楽しめるワークショップ案を提示します。
- **思い出デジタルスクラップ**: その日の写真や動画をアップロードすると、AIが子どもの成長ポイント（「初めて一人で滑り台を滑った！」など）をキャッチアップし、素敵な日記を自動生成します。

## 推奨技術スタック
- **フロントエンド**: Flutter (モバイルアプリとして展開)
- **バックエンド**: Python (Django)
- **外部API**: Google Maps API (場所検索), OpenWeatherMap (天気), OpenAI GPT-4o (プラン生成・日記作成)
- **データベース**: Firebase

## ロードマップ
- **v1.0**: 天気と興味に基づいたお出かけスポット提案。
- **v2.0**: 持ち物リスト生成とカレンダー連携。
- **v3.0**: ユーザー同士のコミュニティ機能（「この公園の授乳室は綺麗だった」などのリアルなパパママ情報のシェア）。
