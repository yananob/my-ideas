# AIこども『ことば』の宝箱

## 概要
子どもの「言い間違い」や「独特な表現」、そして「初めて話した言葉」は、その時期だけの貴重な宝物です。AIこども『ことば』の宝箱は、子どもの可愛いおしゃべりを音声やテキストで簡単に記録し、AIがそれを感情豊かに整理・分析するアプリです。成長の記録としてだけでなく、家族で振り返って楽しめるデジタル宝箱を提供し、子どもの言語発達を温かく見守るサポートをします。

## 主な機能
- **AI魔法のボイスメモ**: AIが周囲の雑音を除去し、子どもの声をクリアに抽出。何を言っているか聞き取りにくい時期でも、文脈からAIが「おそらくこう言っています」という推論付きで文字起こしします。
- **「言い間違い」ライブラリ**: 「てれびくん」を「てれびん」と言うような、期間限定の可愛い言い間違いを自動検知してコレクション。その言葉が「正しく言えるようになった日」もあわせて記録し、成長の軌跡を残します。
- **成長語彙グラフ & エピソード抽出**: 話せるようになった言葉の数や種類（動物、食べ物、感情など）を視覚化。また、AIが記録の中から「心温まるエピソード」や「おもしろ発言」を自動でピックアップして要約します。
- **家族への「今日の名言」配信**: 離れて暮らす祖父母などに、その日に記録された「名言」を音声付きのデジタルカード形式で自動送信。家族みんなで子どもの成長を共有できます。

## ロードマップ
- **v1.0**: 高精度な音声録音・文字起こし機能と、日付・年齢に基づいたシンプルな保存・再生機能。
- **v2.0**: 言い間違いの自動抽出アルゴリズムと、語彙のカテゴリー別分析レポート機能。
- **v3.0**: 記録した言葉を元にした「パーソナライズ成長物語」の自動生成と、思い出の動画・写真との自動マッチング機能。
