# AIわが家のレシピ手帖

## 概要
日々の食事記録から「わが家の定番」と「家族の好み」を学習し、飽きのこない献立を提案してくれるパーソナルな料理エージェントです。何を食べたかを記録するだけで、最近食べていないお気に入り料理のリマインドや、1週間分の献立作成をサポート。ユーザーとの対話を通じて、その日の気分に合わせた柔軟な調整も可能です。

## 主な機能
- **「わが家の味」学習機能**: 食べた料理を記録することで、家族のお気に入りのメニューや味の好みをAIが学習。使い続けるほど、家族の「好き」が詰まった独自のデータベースが構築されます。
- **ご無沙汰・リピート提案**: 「最近これを食べていないね」という、以前好評だった定番メニューを絶妙なタイミングで提案。献立のマンネリ化を防ぎ、家族の満足度を高めます。
- **1週間の献立オート作成**: 過去の履歴と好みのバランスを考慮し、1週間分の献立プランを自動で作成します。もう「今日の晩ごはん、何にしよう？」と悩む必要はありません。
- **対話型献立アレンジ**: 提案された献立に対して、「今日はもっと軽くしたい」「冷蔵庫にあるキャベツを使いたい」といったリクエストを伝えるだけで、AIが即座に献立を組み直します。

## ロードマップ
- **v1.0**: 食事記録機能と、お気に入り料理の自動抽出・リスト化。
- **v2.0**: 履歴に基づいた「ご無沙汰レシピ」のレコメンドと、1週間の献立作成機能。
- **v3.0**: チャット形式でのリアルタイムな献立調整（アレンジ）機能の実装。
