# AIわが家のレシピ手帖

## 概要
日々の料理記録から「わが家の味」を学習し、使えば使うほど家庭の好みにぴったりの献立を提案してくれる、パーソナルなレシピ管理・提案エージェントです。在庫管理などの機能ではなく、家庭ごとの味の継承と、飽きのこない食卓づくりに特化しています。

## 主な機能
- **「わが家の味」学習機能**: 日々のレシピ入力や食べた感想から、家族の味の好み、定番の味付け、お気に入りの食材をAIが深く分析します。
- **ご無沙汰レシピのリマインド**: 「最近これ食べてないね」という、以前好評だったわが家の定番レシピを絶妙なタイミングで提案。献立のマンネリ化を防ぎます。
- **成長に合わせた味付け提案**: 子どもの成長や家族の健康状態に合わせて、いつものレシピを少しアレンジする提案を行います。
- **思い出の味ログ**: 料理の写真とともに、その時の家族の反応やエピソードを記録。単なるレシピ集ではなく、家庭の歴史を刻む「食の記録」となります。

## ロードマップ
- **v1.0**: 日々のレシピ記録機能と、基本的な好み分析の実装。
- **v2.0**: 過去の記録に基づいた「ご無沙汰リマインド」アルゴリズムの搭載。
- **v3.0**: 季節やイベントに合わせた、わが家流の行事食プランニング機能。
