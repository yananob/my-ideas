# AIわが家の『小さな農園』冒険記

## 概要
ベランダのプランターや小さな庭での菜園作りを、家族みんなで楽しむ「命の冒険」に変えるアプリです。AIが植物の状態を分析し、「植物の声」として家族に語りかけることで、子どもの責任感を養い、自然の不思議への好奇心を刺激します。収穫までの道のりをクエスト形式で進めることで、食育と家族のコミュニケーションを同時に実現します。

## 主な機能
- **しゃべる植物バディ**: 植物の写真を撮ると、AIがその健康状態や感情（喉が渇いた、太陽が眩しいなど）を解析。「お水のおかわり、ちょうだい！」といったメッセージが届き、子どもが植物と対話しているような体験を提供します。
- **菜園クエスト & レベルアップ**: 「最初の芽を見つけよう」「本葉が5枚になったら追肥をしよう」といったタスクがクエストとして発行されます。クリアするごとに菜園のレベルが上がり、デジタルバッジや「野菜図鑑」のページが埋まっていきます。
- **AIドクターの往診**: 葉っぱの色がおかしかったり、虫がいたりしたとき、写真を送るとAIが原因と対策をアドバイス。「この虫は味方だよ（天敵）」「この病気にはお酢のシャワーが効くよ」といった具体的な解決策を教えます。
- **はじめての収穫祭プロデュース**: 収穫時期が近づくと、AIがその野菜を使った簡単レシピや、家族でのお祝いパーティのアイデア（メニュー表作り、飾り付けなど）を提案。育てる苦労と収穫の喜びを家族で分かち合う演出をサポートします。

## ロードマップ
- **v1.0**: 写真からの植物特定と、基本的な水やりリマインド、成長記録（写真保存）機能。
- **v2.0**: AIによる健康診断（病害虫検知）機能と、成長段階に合わせた「しゃべるバディ」対話エンジンの実装。
- **v3.0**: 収穫物を使ったパーソナライズレシピ提案と、近隣の「家庭菜園仲間」とアドバイスを交換できるコミュニティ機能。
