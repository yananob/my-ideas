# AI子どもお手伝いクエスト

## 概要
日常の「お手伝い」を壮大なRPGのクエストに変え、子どもが楽しみながら家事に参加できるアプリです。AIが家事をエンターテインメントに変換し、家族全員で楽しめる体験を提供します。

## 主な機能
- **クエスト変換**: 「お部屋の片付け」を「散らかりモンスターの討伐」に、「靴を揃える」を「魔法の陣の整理」に、AIがわくわくする表現に自動変換します。
- **冒険の記録**: お手伝いを達成すると、AIがその活躍を物語風に記録。勇者の成長物語として後から読み返すことができます。
- **AI生成報酬**: お手伝い完了の写真を送ると、AIがそれを解析し、モンスターを倒した瞬間の合成画像や、新しい装備アイテムのイラストを生成してプレゼントします。
- **レベルアップとスキル**: 継続することで「お掃除スキル」や「お料理魔法」のレベルが上がり、子どもの自信と自立心を育みます。

## 推奨技術スタック
- **フロントエンド**: Next.js + Tailwind CSS (モバイルフレンドリーなWebアプリ)
- **バックエンド**: Python (FastAPI)
- **画像生成**: OpenAI DALL-E 3 (報酬画像・アイテム生成)
- **LLM**: OpenAI GPT-4o (クエスト変換、物語生成)
- **データベース**: Supabase (ユーザーデータ、クエスト履歴の保存)

## ロードマップ
- **v1.0**: 基本的なクエスト変換機能とXP（経験値）管理。
- **v2.0**: AIによる報酬画像・アイテムイラスト生成機能の実装。
- **v3.0**: 家族対抗ランキングや、兄弟でのパーティー編成（協力プレイ）機能。
