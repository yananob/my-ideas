# AIこども『お片付け』レンジャー

## 概要
AIの画像認識を活用して、子どもたちが楽しみながらおもちゃの片付けを習慣化できるアプリです。散らかった部屋を「冒険のフィールド」に見立て、おもちゃを元の場所に戻すプロセスをゲーミフィケーション化することで、自発的なお片付けを促します。

## 主な機能
- **お宝スキャン**: 片付けの前に、部屋に散らばったおもちゃをスマートフォンでスキャン。AIがおもちゃを「お宝」として認識し、片付けミッションを開始します。
- **収納マスターガイド**: どのおもちゃをどこに戻せばいいか、AIが画像でアドバイス。正しく片付けられると、AIキャラクターがエフェクトと共に褒めてくれます。
- **ビフォー・アフター表彰**: 片付け前と後の写真をAIが比較。どれだけ綺麗になったかを「ピカピカ度」として数値化し、達成感を可視化します。
- **お片付けレベルアップ**: 毎日の片付け記録を蓄積。頑張りに応じてAIキャラクターが成長したり、新しい「称号」やデジタルアイテムが手に入るゲーミフィケーション要素。

## ロードマップ
- **v1.0**: 画像認識によるおもちゃの検知と、基本的な片付けミッション・表彰機能。
- **v2.0**: 収納場所の写真を事前に登録し、より正確なナビゲーションを可能にする機能。
- **v3.0**: 兄弟や家族で協力して大きな片付けに挑む「チームミッション」モードと、整理整頓のコツを学べる「お片付けクイズ」の実装。
