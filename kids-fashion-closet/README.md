# AIこどもファッション・クローゼット

## 概要
子どもの成長は驚くほど早く、気づけば「お気に入りの服がもう入らない」「同じようなズボンをまた買ってしまった」ということがよくあります。AIこどもファッション・クローゼットは、スマホで服を管理し、AIがサイズアウトの予測や毎日のコーディネートを提案することで、忙しいパパ・ママをサポートし、子どもの自立心を育むアプリです。

## 主な機能
- **AIサイズ管理 & 成長アラート**: 購入した服の写真を撮るか、オンラインショップの購入履歴を連携すると、AIがサイズと種類を自動登録。子どもの身長・体重データと照らし合わせ、「このブランドの110cmは来月には小さくなります」といった予測を通知します。
- **お天気・行事連動コーデ提案**: 朝、その日の天気予報や「今日は遠足」「今日は習い事」といったスケジュールに基づき、クローゼットの中から最適な服装を提案。子どもがスマホやタブレットで「今日の服」を選べる楽しいインターフェースを提供します。
- **おさがり最適化マネージャー**: 兄弟姉妹の年齢差を考慮し、上の子がサイズアウトした服を「下の子の予備」として管理。また、友人知人と「おさがり」を共有できるプライベートなデジタルフリマ/譲渡機能。
- **成長ファッション・アルバム**: 子どもが着ている服を記録し、1年前、2年前の姿と比較。服を通じて子どもの成長を実感できるフォトジャーナル機能。

## ロードマップ
- **v1.0**: 写真からの衣類カテゴリ・サイズ自動認識と、シンプルな在庫リスト機能。
- **v2.0**: 天気連動のコーディネートレコメンドと、子どもの成長予測アルゴリズムの実装。
- **v3.0**: オンラインショップとの連携による不足アイテムの自動提案、およびクローズドな「おさがり」ネットワーク機能。
