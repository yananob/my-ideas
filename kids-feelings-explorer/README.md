# AIこども『きもち』探険隊

## 概要
子どもが自分の感情（怒り、悲しみ、不安など）を正しく理解し、健やかに表現することを助けるAIエージェントです。キャラクターとの対話を通じて、感情を「言葉」や「体のアクション」に変える練習を行い、子どものレジリエンス（心の回復力）を育みます。

## 主な機能
- **きもちの「見える化」パレット**: 今の気分をキャラクターの「色」や「形」で表現。抽象的な感情を可視化することで、子どもが自分の状態を客観的に捉えられるようにします。
- **おたすけ呼吸・ポーズ**: 怒りやパニックを感じたとき、AIキャラクターが一緒に深呼吸や「ひえひえポーズ（落ち着くポーズ）」を画面越しに行い、クールダウンをサポートします。
- **「もやもや」ボイスメッセージ**: 言葉にできないモヤモヤした気持ちをAIに話すと、AIが「それは〜な気持ちかもしれないね」と優しく名前をつけ、共感のメッセージを返します。
- **パパママへの「心の天気予報」**: プライバシーに配慮しつつ、子どもの感情の傾向を保護者に共有。「今日は少し不安を感じているようです。寝る前にたくさん抱っこしてあげてください」といった具体的なサポート方法をアドバイスします。

## ロードマップ
- **v1.0**: 感情パレットによる入力と、感情に応じた励まし・共感メッセージ機能。
- **v2.0**: 音声対話による感情の言語化サポートと、リラックスのための呼吸法ガイドの実装。
- **v3.0**: 感情の記録に基づいた「心の成長日記」生成と、専門家監修による教育コンテンツの拡充。
