# AIスクールプリント・オーガナイザー (AI School Print Organizer)

## 概要
学校や保育園から配布される膨大な紙のプリントを、カメラで撮るだけでAIが自動整理・スケジュール化し、家族で共有できるアプリです。「持ち物忘れ」や「提出期限忘れ」をなくし、忙しいパパ・ママの脳内負荷を軽減します。

## 主な機能
- **AI OCR & カレンダー連携**: プリントを撮影すると、AIがイベント名、日時、場所、持ち物を自動抽出。GoogleカレンダーやTimeTreeなどのカレンダーアプリに自動登録します。
- **持ち物・ToDo自動リスト化**: 「遠足の持ち物」「提出書類」などを自動でチェックリスト化。前日にプッシュ通知でリマインドします。
- **家族間リアルタイム共有**: 登録した情報は即座に家族のデバイスと同期。パパもママも「明日の準備、何が必要だっけ？」と迷うことがなくなります。
- **高度な検索機能**: 「去年の運動会のプリントどこだっけ？」といった時も、キーワードや時期で簡単に検索・閲覧可能です。

## ロードマップ
- **v1.0**: プリントの撮影・OCR機能、日付の自動抽出とカレンダー連携、簡易的な画像保存。
- **v2.0**: 持ち物リストの自動生成、プッシュ通知によるリマインド機能、家族共有機能。
- **v3.0**: 過去の傾向分析（昨年の今頃はこれが必要だった等の提案）、地域の行事カレンダーとの自動照合。
